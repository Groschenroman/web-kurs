Editors

Websites are basically HTML files and as such text files that are written in plain simply text editors.

To write your own HTML file, any editor will do as long as they are pure text editors. (Don't use MS Word, Open Office or the like)

  • 📝 Sublime Text
  • 📝 Atom (by Github, which belongs to Microsoft)
  • 📝 Visual Studio Code (Microsoft)
  • 📝 Notepad++ (Microsoft)

Create a new file called index.html, copy-paste the content below, save the file and double-click it to open it in your default browser
